# Price Intelligence Frontend

Premium Angular 17 dashboard for e-commerce market intelligence in Morocco.

## Strategic Features
- **Global Inventory**: Real-time access to thousands of products across Jumia, Avito, Amazon, and Connecto.
- **Statistical Analysis**: Deep-dive into market variance, ANOVA tests, and confidence intervals.
- **Predictive Modeling**: Visual representation of OLS regression models and price residuals.
- **Market Velocity**: Tracking and visualization of the fastest price drops in the market.
- **Glassmorphism UI**: High-fidelity SaaS aesthetic designed for corporate and academic presentations.

## Technical Stack
- **Framework**: Angular 17 (Standalone Components)
- **Visuals**: Plotly.js for high-fidelity statistical charts.
- **Icons**: Bootstrap Icons & FontAwesome.
- **Styling**: Vanilla CSS with advanced CSS Variables for theme management.

## Development
```bash
npm install
npm start
```
Navigate to `http://localhost:4200/`.
